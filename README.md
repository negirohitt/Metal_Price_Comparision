# Metal_Price_Comparision
# 🪙 Metal Price Comparison

A web-based application that allows users to compare real-time prices of different metals like **Gold, Silver, and more** through an interactive and user-friendly interface.

---

##  Features

* 📊 Compare prices of multiple metals
* 🔄 Real-time / dynamic data handling
* 📈 Clean and interactive UI
* ⚡ Fast and responsive design
* 🧩 Modular frontend structure

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* (Optional: React / Vite if you used it)
* REST APIs (for fetching metal prices)

## ⚙️ How to Run

1. Clone the repository

   ```
   git clone https://github.com/negirohitt/Metal_Price_Comparision.git
   ```

2. Open the project folder

3. Run `index.html` in your browser
   *(or use Live Server in VS Code)*
